About My Rust Server

What is this server?

This is a simple static file server built with the Rust programming language. It serves static files from the "static" folder on the server, making it an ideal solution for serving web assets like HTML, CSS, JavaScript, and images.

Why Rust?

Rust is a systems programming language known for its speed, memory safety, and reliability. It's perfect for building high-performance servers like this one!

How to Use

Simply upload your static files to the "static" directory and visit the appropriate URL to view them. For example, to view an image, place it in the static folder and access it directly in the browser.

Get Involved

If you'd like to contribute to the project, feel free to open an issue or pull request on the project's GitHub repository. We'd love to have you!
